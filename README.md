# TasksGB
## Описание классов
* Comparator.cs - класс "*сравниватель*", который может сравнивать от двух до трёх введённых чисел.
* EvenRange.cs - класс, который может определять и выводить на экран все чётные числа от 0 до введённого числа.
* OdderOrEvener.cs - класс, который может определять чётность или нечётность введённого числа.
* Program.cs - главный класс.
